Тестовое задание на позицию HTML-верстальщик.

– Адаптивная верстка с минимальным разрешением 320px;
– HTML5, CSS, JavaScrip;
– SASS в диалекте SCSS, наименования по методологии БЭМ, используется сборщик Vite;
– Изображения оптимизированы по весу;
– header прилипает к верхней границе и меняет высоту при скроле на экранах шире 1024зч;
– При разрешении меньше 768px навигация реализована в формате бургера;

Проект выполнен в соответствии с ТЗ.
